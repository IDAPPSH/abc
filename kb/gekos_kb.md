# GEKOS – Sistema de Auto Agenda  
## IDAPPSH – Hybrid Systems Lab

Este documento contiene los **Términos y Condiciones del servicio GEKOS**, en su versión resumida para aceptación digital y uso informativo por parte del asistente IA de IDAPPSH.

GEKOS es un sistema propietario desarrollado por IDAPPSH para automatización de búsqueda y gestión de citas bajo diferentes modalidades.

---

## 1. Aceptación del servicio

Al aceptar estos términos, el usuario autoriza expresamente la **instalación, configuración y ejecución** del sistema:

- GEKOS  
- GEKOS Retro  
- GEKOS Full  
- GEKOS Classic  
- GEKOS Guardian  

El uso del sistema implica la **aceptación total** de este documento.

---

## 2. Política de pagos

El usuario reconoce y acepta que el servicio GEKOS opera bajo el siguiente esquema:

- **Pago inicial único** por instalación, configuración y capacitación  
  - Este pago es **no reembolsable**.
- **Mensualidad recurrente** por:
  - Uso del sistema
  - Actualizaciones
  - Componentes de IA
  - Soporte técnico
- **Servicios adicionales** (por ejemplo, alertas especiales) pueden tener un costo independiente.

---

## 3. Garantía de cita

Si durante el ciclo contratado **NO se genera ninguna cita real válida** para el perfil configurado:

- La mensualidad **NO se corta**
- El servicio **NO se suspende**
- **NO se vuelve a cobrar** hasta que se obtenga **al menos una cita válida**

Una vez obtenida la cita válida:
- Se inicia un **nuevo ciclo de facturación**
- La fecha de referencia se actualiza al momento de la cita obtenida

### Esta garantía aplica únicamente si:
- El perfil es técnicamente agendable
- Existe disponibilidad real en el/los consulados configurados
- El usuario **no ha manipulado, bloqueado o alterado** credenciales o configuración

---

## 4. Limitaciones del sistema

El usuario entiende y acepta que:

- La disponibilidad de citas depende **exclusivamente** del portal oficial del consulado o embajada
- IDAPPSH **NO controla**:
  - Horarios
  - Cupos
  - Cambios unilaterales de plataformas externas
- El sistema GEKOS no garantiza disponibilidad cuando el portal externo no la ofrece

---

## 5. Responsabilidades del usuario

El usuario se compromete a:

- Mantener su equipo estable y con conexión adecuada a Internet
- No cerrar, interrumpir ni interferir con los procesos del sistema
- No modificar contraseñas ni datos críticos sin notificar y reconfigurar
- No compartir credenciales con terceros sin autorización expresa

---

## 6. No reembolso del pago inicial

El pago inicial cubre:

- Instalación
- Ajuste de entorno
- Configuración
- Capacitación
- Tiempo técnico invertido

Por lo tanto, **es estrictamente no reembolsable** bajo cualquier circunstancia.

IDAPPSH se reserva el derecho de **cancelar el servicio sin reembolso** cuando:
